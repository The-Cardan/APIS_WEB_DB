# APIS_WEB_DB
Este repositorio es para la práctica 3 y 4, una biblioteca necesita un sistema para gestionar su catálogo de libros. Trata de una API REST en ASP.NET Core que permita consultar, registrar, actualizar y eliminar libros, así como obtener el catálogo en forma paginada. La información accede a  una base de datos SQL Server.
